Esta carpera "Languages" con todas las traduccciones (ficheros con extensión .lng) debe copiarse en la misma ubicación de la aplicación.
Al arrancar la aplicación, leerá automáticamente todos los ficheros de traducción disponibles y generará un menú de forma dinámica con todos los idiomas disponibles.

This "Languages" folder, containing all translations (files with the .lng extension), must be copied to the same location as the application. 
When the application starts, it will automatically read all available translation files and dynamically generate a menu with all available languages.
